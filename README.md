# manumental-effort

A real-time chat platform where men and people who like men can connect, talk, and support each other through shared spaces.

## Current Status
Early architecture and bootstrap phase.

## Core Stack
- Backend: Go + Gin
- Frontend: React
- Database: MongoDB
- Runtime: Linux Mint for v1
